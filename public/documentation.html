Project Documentation

Project Title

myFlix

Project Description

This project sets up the server side of an API that provides information on different movies, directors, and genres. Users will be able to sign up for the application, update their personal information, and create a lists of movies.

This project uses REST architecture.

URL Endpoints and Request Methods
